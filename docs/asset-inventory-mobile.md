# Asset Inventory - Mobile

## Overview
Assets bundled with the Ionic mobile application.

## Images (`src/assets/images/` or `src/assets/icon/`)
*   `icon.png`: App icon (adaptive).
*   `splash.png`: Splash screen image.
*   `logo.png`: App logo for login screen.
*   `empty-state.svg`: Illustration for empty lists.
*   `offline.svg`: Illustration for offline status.

## Configuration
*   `capacitor.config.ts`: Native platform configuration (Bundle ID, Plugins).
*   `ionic.config.json`: Ionic CLI settings.
*   `android/`: Native Android project files (Gradle, Manifest).
*   `ios/`: Native iOS project files (Info.plist, Xcode project).

## Resources
*   `src/theme/variables.scss`: Ionic global style variables.
*   `src/global.scss`: Global CSS rules.
