# ELYKIA Project Documentation

## Project Documentation Index

### Project Overview
*   **Type:** Monorepo with 3 parts
*   **Primary Languages:** Java, TypeScript
*   **Architecture:** Distributed (Backend + Web + Mobile)

### Quick Reference

#### Backend Service (backend)
*   **Type:** Backend
*   **Tech Stack:** Spring Boot 3.3, Java 17, PostgreSQL
*   **Root:** `backend/`

#### Web Frontend (frontend)
*   **Type:** Web
*   **Tech Stack:** Angular 14, TypeScript
*   **Root:** `frontend/`

#### Mobile App (mobile)
*   **Type:** Mobile
*   **Tech Stack:** Ionic 8, Angular 20, Capacitor 7
*   **Root:** `mobile/`
