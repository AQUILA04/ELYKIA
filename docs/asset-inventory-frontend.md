# Asset Inventory - Frontend

## Overview
Static assets used in the Angular frontend application.

## Images (`src/assets/images/`)
*   `logo.png`: Main application logo.
*   `logo-white.png`: Logo for dark backgrounds/sidebar.
*   `favicon.ico`: Browser tab icon.
*   `login-bg.jpg`: Background image for the login page.
*   `placeholder-user.png`: Default avatar for users.

## Icons
*   **Material Icons:** Loaded via web font (Google Fonts).
*   **FontAwesome:** Used for specific social or action icons not found in Material.

## Styles (`src/assets/scss/` or `src/styles.scss`)
*   `_variables.scss`: Global color palette, spacing, and typography variables.
*   `_mixins.scss`: Reusable SCSS mixins.
*   `theme.scss`: Angular Material theme customization.

## Configuration
*   `src/assets/i18n/`: (If applicable) JSON files for translations (en.json, fr.json).
