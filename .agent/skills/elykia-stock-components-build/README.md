# ELYKIA Stock Module Components Build Skill

Ce skill est conçu pour les agents de développement (comme Manus) travaillant sur le frontend mobile du projet ELYKIA, spécifiquement pour le module de gestion des stocks.

## Objectif

L'objectif de ce skill est de fournir un guide unifié et prescriptif pour la construction de l'interface utilisateur du module de gestion des stocks. Il traduit les spécifications visuelles (Design System, charte graphique, maquettes) en règles techniques concrètes et réutilisables pour le framework Ionic/Angular.

## Utilisation

Ce skill doit être lu par tout agent chargé de développer, modifier ou réviser des composants UI pour les écrans de gestion des stocks (liste des demandes, détails d'une demande, création de demande).

Les agents doivent :
1.  Lire le fichier `SKILL.md` pour une vue d'ensemble.
2.  Consulter les fichiers spécifiques dans le dossier `rules/` pour des directives détaillées sur les tokens de design, le styling, les composants clés et l'accessibilité.
3.  Appliquer strictement ces règles lors de l'écriture du code HTML, SCSS et TypeScript.

## Contenu

Le skill est organisé autour des thèmes suivants :
*   **Design Tokens :** Palette de couleurs, typographie, formes et élévation.
*   **Styling :** Approche Mobile-First, variables CSS Ionic, et gestion des états de chargement (Skeletons).
*   **Composants UI :** Structure et implémentation des composants partagés (Header, Badges, Cartes, Répéteur d'articles).
*   **Accessibilité (A11y) :** Règles pour les zones de clic, le contraste, les icônes et les formulaires.
