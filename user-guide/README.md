# Documentation Utilisateur AMENOUVEVE-YAVEH

Ce dossier contient les sources du guide utilisateur généré avec [MkDocs](https://www.mkdocs.org/).

## Pré-requis

*   Python installé.
*   MkDocs et le thème Material installés.

## Installation

Ouvrez un terminal dans ce dossier et exécutez :

```bash
pip install mkdocs mkdocs-material
```

## Commandes

### Prévisualisation en direct
Pour voir le site en local avec rechargement automatique :

```bash
mkdocs serve
```
Le site sera accessible sur `http://127.0.0.1:8000`.

### Génération du site statique (Build)
Pour générer les fichiers HTML (dossier `site/`) prêts à être déployés :

```bash
mkdocs build
```
