# Reporting & Synthèse

Le module de rapports permet d'avoir la situation financière exacte en temps réel.

## 📊 Consultation des Rapports

### 1. Accéder aux Rapports
Dans le menu latéral, cliquez sur **Rapports**.

### 2. Choisir une Période
Par défaut, le système peut ne rien afficher ou afficher la journée en cours.
1.  En haut à gauche, cliquez sur la liste déroulante **Période**.
2.  Choisissez l'intervalle que vous voulez analyser (Ex: *Aujourd'hui*, *Hier*, *Ce Mois*...).
3.  Cliquez sur le bouton avec la **Loupe** (`search`) juste à côté pour lancer le calcul.

### 3. Lire le Bilan Global (En haut)
Une fois la recherche lancée, trois informations capitales s'affichent en haut :
*   **DU ... AU ...** : Confirme les dates prises en compte.
*   **Total recouvrement** : C'est toute l'argent qui est entré dans les caisses (Ventes + Tontines).
*   **Total sorties** : C'est la valeur de la marchandise qui a été livrée aux clients.

### 4. Analyser le Tableau par Commercial (Au centre)
Le tableau vous donne le détail pour chaque agent. C'est l'outil de contrôle principal.

Colonnes :
*   **Noms de l'agent** : Qui est concerné ?
*   **Montant système** : Ce que l'ordinateur a calculé. C'est la somme théorique des opérations saisies.
*   **Montant réel** : Ce que le commercial a physiquement dans sa caisse (déclaré lors de la fermeture ou du billetage).
*   **Écart** (Dernière colonne) : C'est la différence entre le Réel et le Système.
    *   Si le chiffre est sur fond **Vert** : Il y a un surplus (Plus d'argent que prévu).
    *   Si le chiffre est sur fond **Rouge** : Il y a un manquant (De l'argent a disparu).
    *   Si le chiffre est **0** : La caisse est parfaite.

!!! tip "Action Recommandée"
    Si vous voyez une case Rouge, cliquez sur le bouton **Rafraîchir** pour être sûr d'avoir les dernières données, puis contactez le commercial concerné pour justification.
